Dette prosjektet er en fullstack webapplikasjon utviklet i Node.js som lar kunder sende forespørsler om IT-hjelp.
Henvendelser lagres automatisk og kan administreres via et eget admin-panel med innlogging.

Målet med prosjektet er å koble kunder med IT-tjenester – en enkel start på en lead generation business.


⚙️ Funksjonalitet
📝 Kunde kan sende forespørsel via skjema
💾 Data lagres automatisk i JSON-database (kunder.json)
🔐 Admin login system
📊 Admin-side for å se alle kunder



🛠️ Teknologi
Node.js
Express.js
HTML / CSS / JavaScript
JSON (database)
Nodemailer (email)
